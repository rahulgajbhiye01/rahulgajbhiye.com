## 💻 Code Blocks with Syntax Highlighting

### JavaScript Example

```javascript
function greet(name) {
  return `Hello, ${name}! Welcome to enhanced markdown!`;
}

// Call the function
console.log(greet("Developer"));
```

### Python Example

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Generate first 10 Fibonacci numbers
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```
